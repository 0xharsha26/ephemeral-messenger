# 🔐 Ephemeral Messenger

A secure real-time encrypted messaging system with self-destructing messages.

## Features
- 🔐 Client-side encryption (AES-GCM)
- ⚡ Real-time messaging (WebSockets)
- ⏳ Self-destructing messages
- 🎨 Custom UI (Electric Purple theme)
- 🔑 Invite-based access system

## Tech Stack
- FastAPI
- WebSockets
- SQLModel
- Web Crypto API
- JavaScript

## Future Improvements
- Public key encryption
- ESP32 integration
- Auto cleanup system
